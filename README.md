# Retail Sales Analysis (SQL Project)

## Project Overview

This project analyzes retail sales data to understand store performance, product demand, and sales trends using SQL.

## Database Tables

The database consists of four main tables:

* stores
* products
* transactions
* transaction_details

## Business Questions

The analysis answers the following business questions:

1. Which store generates the highest sales?
2. What is the total revenue?
3. What is the average transaction value?
4. What are the daily sales trends?
5. What are the peak sales hours?

## Key Insights

* Jakarta Store generated the highest revenue.
* Total revenue reached 1,965.
* The average transaction value is 491.25.
* Sales activity peaks around mid-day hours.

## Tools Used

* Microsoft SQL Server
* SQL (JOIN, GROUP BY, Aggregation)
